
C Programming Mini Projects
This repository contains three simple C programs for beginners:

Temperature Converter (Celsius to Fahrenheit)
Gross Salary Calculator
Triangle Angle Finder
⚠️ Note: Each program has its own main() function, so they must be saved and compiled separately in different .c files.

1. Temperature Converter (Celsius to Fahrenheit)
File name: temperature_converter.c

Description
This program converts temperature from Celsius to Fahrenheit using the formula: F = (9/5 × C) + 32

Gross Salary Calculator
File name: gross_salary.c

Description

This program calculates the gross salary based on:

Base Salary

HRA percentage

DA percentage

TA percentage

Triangle Angle Finder
File name: triangle_finder.c

Description

This program finds the third angle of a triangle when the first two angles are given.

Formula used: Third Angle = 180 − (First Angle + Second Angle)
